# HerSafe-Network
# 🌐 HerSafe Network (Web MVP)

A community-driven safety platform designed to support **solo women travelers** by connecting them with **trusted local guardians** in real-time.

---

## 🚀 Overview

HerSafe Network is a web-based MVP that aims to reduce uncertainty and improve safety for women traveling alone. Instead of relying only on emergency services, the platform builds a **human + technology safety layer**, enabling travelers to quickly connect with verified locals nearby.

---

## 🧠 Problem

Solo women travelers often face:

* Safety concerns in unfamiliar areas
* Lack of immediate trusted support
* Difficulty identifying safe routes or people
* Delayed response in emergency situations

Existing solutions are **reactive** (SOS, helplines) rather than **proactive and community-driven**.

---

## 💡 Solution

HerSafe Network introduces:

* A **guardian network** of verified locals
* Real-time **chat and SOS system**
* A **trust-based verification model** (without complex KYC)

The goal is simple:

> Make travelers feel like they are **not alone in any city**

---

## ✨ Features (MVP)

### 🔐 Authentication

* Phone number login (OTP-based)
* Role selection: Traveler / Guardian

---

### 🛡️ Guardian Verification (No KYC)

* Selfie-based identity check
* Optional social media linking
* Trust levels:

  * Basic Verified
  * Trusted

---

### 📍 Traveler Dashboard

* View nearby guardians
* Access guardian profiles
* Request help via chat

---

### 🧑‍🤝‍🧑 Guardian Dashboard

* Toggle availability (Available / Busy)
* Accept or decline help requests

---

### 💬 Real-Time Chat

* 1-to-1 messaging
* Optional location sharing

---

### 🚨 SOS System

* One-tap emergency alert
* Sends location to nearby guardians
* Notifies multiple guardians simultaneously

---

### ⭐ Rating & Reviews

* Travelers can rate guardians
* Builds a community trust layer

---

### 🗺️ Basic Safety Map

* Displays nearby guardians
* Approximate location visibility

---

## 🏗️ Tech Stack

* **Frontend:** React / No-code builder (Bubble, FlutterFlow)
* **Backend:** Firebase (Auth, Firestore, Realtime DB)
* **AI (optional):** DeepSeek (via OpenRouter)
* **Maps:** Google Maps API
* **Face Verification (optional):** FaceIO / AWS Rekognition

---

## 🔄 How It Works

1. User signs up via OTP
2. Chooses role (Traveler / Guardian)
3. Traveler browses nearby guardians
4. Starts chat or sends SOS
5. Guardian responds and helps
6. Traveler rates the experience

## ⚠️ Disclaimer

* This platform does **not guarantee safety**
* Guardians are **community-verified**, not government verified
* Users should always use personal judgment and local emergency services when needed
